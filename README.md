## How to Run

[Setup an AWS Account](https://aws-amplify.github.io/docs/) 

Install amplify cli globally
### `npm install -g @aws-amplify/cli`

Configure a user in your AWS account
### `amplify configure`
 
Install amplify library and react helper library
### `yarn add aws-amplify aws-amplify-react`

Add hosting to the app
### `amplify add hosting`

Add an API for the app
### `amplify add api`

Push your changes to AWS
### `amplify push`
